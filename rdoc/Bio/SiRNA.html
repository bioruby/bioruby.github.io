    class Bio::SiRNA

    Bio::SiRNA¶ ↑

    Designing siRNA.

    This class implements the selection rules described by Kumiko Ui-Tei et al. (2004) and Reynolds et al. (2004).

    Attributes

    antisense_size [RW]

    A parameter of size of antisense.

    max_gc_percent [RW]

    A parameter of maximal %GC.

    min_gc_percent [RW]

    A parameter of minimum %GC.

100   seq19 = target[2..20]  # 19 nt double-stranded region of siRNA
101   score = 0
102 
103   # criteria I
104   gc_number = seq19.scan(/[GC]/i).size
105   score += 1 if (7 <= gc_number and gc_number <= 10)
106 
107   # criteria II
108   au_number = seq19[14..18].scan(/[AU]/i).size

111   # criteria III
112   # NotImpremented: Tm
113
