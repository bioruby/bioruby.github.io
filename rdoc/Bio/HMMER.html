    class Bio::HMMER

    Description¶ ↑

    A wapper for HMMER programs (hmmsearch or hmmpfam).

    Examples¶ ↑

    require 'bio'
program = 'hmmsearch' # or 'hmmpfam'
hmmfile = 'test.hmm'
seqfile = 'test.faa'

factory = Bio::HMMER.new(program, hmmfile, seqfile)
report = factory.query
report.class # => Bio::HMMER::Report

    References¶ ↑

    • HMMER hmmer.wustl.edu/
