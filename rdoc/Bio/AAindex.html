    auto(str) click to toggle source

    Auto detecter for two AAindex formats. returns a Bio::AAindex1 object or a Bio::AAindex2 object.

    # File lib/bio/db/aaindex.rb, line 57
def self.auto(str)
  case str
  when /^I /m 
    Bio::AAindex1.new(str)
  when /^M /m
    Bio::AAindex2.new(str)
  else
    raise
  end        
end
